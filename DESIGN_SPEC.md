# AI 批量搜索 · UI 设计规范

> 本文件记录项目当前的设计 Token 与组件用法规范，所有新增 UI 应严格遵守。
> 最后更新：2026-04-16

---

## 一、字体

### 字族

```css
font-family: "Segoe UI", Arial, sans-serif;
```

Windows 首选 Segoe UI，降级用 Arial，最终保底 sans-serif。

### 字号 Token

| Token      | 值     | 用途                               |
|------------|--------|------------------------------------|
| `--fs-sm`  | 12px   | 辅助信息、标签、时间戳、Tooltip     |
| `--fs-md`  | 14px   | 正文、按钮、输入占位符、列表项      |
| `--fs-lg`  | 16px   | 输入框文字、卡片标题、弹窗标题      |

### 字重规范

| 场景                   | font-weight |
|------------------------|-------------|
| 普通正文               | 400         |
| 按钮、标签、辅助文字   | 500         |
| 列表标题、卡片标题     | 600         |
| 面板标题（h3 级别）    | 700         |

### 行高规范

| 场景             | line-height |
|------------------|-------------|
| 单行 UI 元素     | 1           |
| 标题             | 1.35        |
| 正文内容         | 1.5         |
| 输入框           | 1.45        |

## 二、圆角

### 圆角 Token

| Token     | 值    | 用途                                               |
|-----------|-------|----------------------------------------------------|
| `--r-sm`  | 6px   | Tooltip、小型标签、代码块                          |
| `--r-md`  | 12px  | 普通按钮、标签、下拉菜单项、弹窗内部分组            |
| `--r-lg`  | 18px  | **主要容器**：输入框、卡片主体、历史记录条目、弹窗  |
| `999px`   | 全圆  | 圆形图标按钮、头像、进度条、滚动条滑块              |
| `50%`     | 圆形  | 关闭按钮（正方形元素变圆）                         |

### 使用规则

- **所有卡片级容器**（输入框壳、AI 卡片主体、历史记录条目、弹窗）→ 统一用 `--r-lg`（18px）
- **行内交互元素**（布局按钮、导出格式按钮、站点标签）→ 用 `--r-md`（12px）
- **辅助说明**（Tooltip、小提示框）→ 用 `--r-sm`（6px）
- **图标按钮**（工具栏圆形按钮、卡片操作按钮）→ 用 `999px`

## 三、颜色

### 文字颜色

| Token          | 值        | 用途                           |
|----------------|-----------|--------------------------------|
| `--text-main`  | `#1a1a1a` | 正文、输入内容                 |
| `--text-muted` | `#666`    | 辅助说明、时间戳、占位符       |
| —              | `#111`    | 标题、强调文字、激活态文字     |
| —              | `#444`    | 次级正文、按钮默认文字         |
| —              | `#aaa`    | 禁用/弱化文字、清空按钮默认态  |

### 背景颜色

| 值          | 用途                             |
|-------------|----------------------------------|
| `#ffffff`   | 主页面背景、卡片背景             |
| `#fafafa`   | 条目 hover 背景                  |
| `#f5f5f5`   | 标签默认背景、工具栏按钮 hover   |
| `#fbfbfb`   | 弹窗侧边栏背景                   |
| `#f9fafb`   | 导出弹窗提示条背景               |
| `#f8fafc`   | 代码预览背景                     |

### 边框颜色

| 值                   | 用途                                       |
|----------------------|--------------------------------------------|
| `rgba(0,0,0, .07)`   | 侧边面板左边框（极弱分割线）               |
| `rgba(0,0,0, .08)`   | **主边框**：卡片主体、历史条目             |
| `rgba(0,0,0, .10)`   | 标签、弹窗内元素、普通按钮边框             |
| `rgba(0,0,0, .12)`   | 图标按钮边框                               |
| `rgba(0,0,0, .14)`   | 输入框壳边框（最显眼的主要容器）           |
| `#ddd`               | 布局选项按钮、导出按钮（不透明场合）       |
| `--border-color` `#e0e0e0` | 遗留 token，新增组件优先用上方透明写法 |

> **规律**：边框透明度从 .07 → .08 → .10 → .12 → .14，数值越大边框越明显，对应容器越重要。
> **原则**：全部使用 `rgba(0,0,0,…)` 纯黑透明，禁止使用带蓝色分量的 `rgba(15,23,42,…)`。

### 交互/激活色

| 值        | 用途                                       |
|-----------|--------------------------------------------|
| `#111`    | 发送按钮背景、激活态（最深黑）             |
| `#444`    | 发送按钮 hover                             |
| `#000000` | 布局按钮激活态、站点导航激活态             |
| `#2563eb` | 主操作色（primary，目前较少使用）          |
| `#ef4444` | 删除/危险操作 hover                        |

## 五、动画过渡

| 场景               | transition                                          |
|--------------------|-----------------------------------------------------|
| 通用快速           | `.15s`                                              |
| 卡片 hover         | `box-shadow .18s, border-color .18s, background .18s` |
| 工具按钮 hover     | `background .2s ease, transform .2s ease`           |
| 侧边面板滑入       | `transform .36s cubic-bezier(.25,.46,.45,.94)`      |
| 按钮上浮效果       | `transform: translateY(-1px)`                       |
| Tooltip 显示       | `opacity .12s ease, visibility .12s ease`           |

## 六、核心组件规格速查

### 输入框壳（Composer）

```
border: 1px solid rgba(15,23,42,.14)
border-radius: var(--r-lg)          /* 18px */
background: #fff
box-shadow: 0 16px 38px rgba(15,23,42,.14)
padding: 10px 18px 12px
```

### AI 卡片主体

```
border: 1px solid rgba(15,23,42,.08)
border-radius: var(--r-lg)          /* 18px */
background: #fff
box-shadow: 0 14px 30px rgba(15,23,42,.08)
```

### 历史记录条目

```
border: 1px solid rgba(15,23,42,.08)
border-radius: var(--r-lg)          /* 18px */
background: #fff
box-shadow: 0 2px 8px rgba(15,23,42,.04)
padding: 13px 34px 13px 16px
```

### 站点标签（Tag）

```
height: 26px
padding: 0 10px
border: 1px solid rgba(15,23,42,.1)
border-radius: var(--r-md)          /* 12px */
background: #f3f4f6
color: #374151
font-size: var(--fs-sm)
font-weight: 500
hover → background: #111827; color: #fff
```

### 普通行内按钮（如布局选项）

```
height: 34px
padding: 0 12px
border: 1px solid #d1d5db
border-radius: var(--r-md)          /* 12px */
background: #fff
color: #111827
font-size: var(--fs-md)
font-weight: 500
active → background: #000; color: #fff; border-color: #000
```

### 图标圆形按钮

```
width/height: 34px ~ 42px（视场景）
border-radius: 999px
background: transparent 或 #fff
hover → background: #f3f4f6 或 translateY(-1px)
```

### Tooltip

```
padding: 5px 10px
font-size: var(--fs-sm)
font-weight: 600
color: #fff
background: rgba(15,23,42,.92)
border-radius: var(--r-sm)          /* 6px */
box-shadow: 0 4px 12px rgba(15,23,42,.2)
```

## 七、设计原则总结

1. **圆角越大 = 层级越高**：Tooltip(6) → 按钮/标签(12) → 主容器(18) → 图标按钮(999px)
2. **边框透明度越高 = 容器越重要**：分割线(.07) → 卡片(.08) → 标签(.10) → 输入框(.14)
3. **黑白为主**：激活态、发送按钮统一用 `#111827` / `#000`，不用彩色
4. **彩色只用于危险操作**：删除、清空 hover 用红色 `#ef4444`
5. **悬停用微提升**：hover 时 `translateY(-1px)` 产生轻微上浮感
6. **字重用 500/600/700 三档**：标签500 → 正文标题600 → 面板标题700
